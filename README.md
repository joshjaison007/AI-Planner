**AI-Planner** is a high-performance productivity suite that transforms unstructured natural language into actionable, structured project schedules. Developed under the **Liquid AI** venture, this project was a featured solution at **Lazerhacks 2026** (hosted by HackCC).

---

## Overview
Traditional planning tools are rigid and time-consuming. **AI-Planner** leverages the **Gemini API** to handle the heavy lifting. By simply "brain-dumping" your tasks in plain English, our engine extracts deadlines, prioritizes milestones, and generates a cohesive plan ready for export.

## Key Features
- **Natural Language Orchestration:** Describe your day naturally; the AI handles the scheduling logic.
- **CSV Data Portability:** Export your generated plans to **CSV** (for Excel/Google Sheets) or import existing lists for AI optimization.
- **Cloud Ready:** Direct compatibility for uploading exported schedules to **Google Drive**.
- **Dynamic UX:** Integrated **Light Mode & Dark Mode** for a high-accessibility user experience.
- **Zero-Config Deployment:** Fully automated setup via Windows Batch—no manual installation required.

---

## Quick Start (One-Click Run)

AI-Planner is pre-configured for immediate deployment.

### **Launch Instructions**
1. **Extract:** Download and extract the project Liquid Ai ZIP file to your computer.
2. **Run:** Double-click **`deploy.bat`**. 
3. **Access:** The application will automatically initialize all dependencies and launch in your default browser.
